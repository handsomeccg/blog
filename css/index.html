# BFC

BFC里面的布局不对外面产生影响 作用： 1.清除外部浮动 2.撑开浮动塌陷（浮动元素的高度会被计算进来） 2.清除外边距重叠

如何创建BFC：

  • 根元素（html）
  • 浮动元素
  • 行内块元素
  • 表格单元格
  • overflow不为visible
  • flex布局
  • display: flow-root

# 盒模型

标准盒模型： width + padding + borderWidth
IE盒模型： width (内容区根据计算获得具体大小)
