# Koa

koa洋葱模型：

  • 洋葱模型内的每一层都表示一个独立的中间件，用于实现不同的功能。
  • 每次请求都会从外层开始一层层经过每个中间件，到达最里层的中间件后，再逐层返回。
  • 对于每个中间件来说，在一个请求和响应的周期中，都有两个时机点来添加不同的处理逻辑。

为什么使用洋葱模型： 对后续中间件返回的结果进行处理
